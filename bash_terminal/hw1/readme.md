6. Добавить оперативной памяти или ресурсов процессора можно изменив конфигурацию в файле Vagrantfile

```
   Vagrant.configure("2") do |config|
      config.vm.box = "bento/ubuntu-20.04"  
	  config.vm.provider "virtualbox" do |v|
	     v.memory = 4096
		 v.cpus = 4
	  end	
   end
```
